# Документация API

Документация по публичному API пакета `resilience`.

## Основные классы

- [СтратегияПовтора](СтратегияПовтора.md) (`Retry`) — повторяет операцию при ошибке
- [СтратегияПредохранителя](СтратегияПредохранителя.md) (`Circuit Breaker`) — блокирует вызовы после серии сбоев
- [СтратегияОграниченияПараллелизма](СтратегияОграниченияПараллелизма.md) (`Bulkhead`) — ограничивает количество одновременно выполняющихся операций и размер очереди ожидания
- [СтратегияТаймаута](СтратегияТаймаута.md) (`Timeout`) — ограничивает время ожидания завершения операции
- [СтратегияОграниченияВызовов](СтратегияОграниченияВызовов.md) (`Rate Limiter`) — ограничивает частоту вызовов
- [СтратегияЗапасногоСценария](СтратегияЗапасногоСценария.md) (`Fallback`) — возвращает запасной результат при ошибке
- [ПайплайнОтказоустойчивости](ПайплайнОтказоустойчивости.md) — собирает стратегии в цепочку выполнения
- [СигналПрерыванияОперации](СигналПрерыванияОперации.md) — сигнал кооперативного прерывания, который можно передавать в пайплайн и стратегии
- [ОбработчикОтказоустойчивости](ОбработчикОтказоустойчивости.md) — подготовка и вызов обработчиков с контекстом и дополнительными параметрами
