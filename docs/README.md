# Документация API

Документация по публичному API пакета `resilience`.

## Основные классы

- [ПайплайнОтказоустойчивости](ПайплайнОтказоустойчивости.md) — композиция нескольких стратегий вокруг одной операции
- [ОбработчикОтказоустойчивости](ОбработчикОтказоустойчивости.md) — подготовка и вызов обработчиков с контекстом и дополнительными параметрами
- [СтратегияПовтора](СтратегияПовтора.md) — `Retry`
- [СтратегияПредохранителя](СтратегияПредохранителя.md) — `Circuit Breaker`
- [СтратегияОграниченияПараллелизма](СтратегияОграниченияПараллелизма.md) — `Bulkhead`, ограничение параллелизма и очереди ожидания
- [СтратегияТаймаута](СтратегияТаймаута.md) — `Timeout`
- [СтратегияОграниченияВызовов](СтратегияОграниченияВызовов.md) — `Rate Limiter`
- [СтратегияЗапасногоСценария](СтратегияЗапасногоСценария.md) — `Fallback`
