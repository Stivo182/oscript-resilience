# ИнтерфейсСтратегииОтказоустойчивости

**Описание:**

Документирующий контракт пользовательской стратегии отказоустойчивости.

Интерфейс не предназначен для прямого использования в пайплайне. Он показывает минимальный публичный API, который должен поддерживать объект пользовательской стратегии для работы с [ПайплайнОтказоустойчивости](ПайплайнОтказоустойчивости.md).

## Методы

[Применить](#применить)

## Применить

**Синтаксис:**

```bsl
Применить(<Операция>, <Параметры>)
```

**Параметры:**

| Имя | Тип | Значение по умолчанию | Описание |
| -- | -- | -- | -- |
| Операция | Действие, ШагПайплайнаОтказоустойчивости, Строка |  | Пользовательская операция или вложенный шаг пайплайна. Строка трактуется как лямбда-выражение операции |
| Параметры | Массив, ФиксированныйМассив, Произвольный, Неопределено | `Неопределено` | Параметры для передачи в пользовательскую операцию |

**Возвращаемое значение:**

Тип: Произвольный.

**Описание:**

Пользовательская стратегия должна реализовать этот метод и вернуть результат выполнения операции или собственное значение, сформированное поверх результата.

Внутри метода рекомендуется использовать `АдаптерОперацииОтказоустойчивости` для подготовки `Действие`, строкового лямбда-выражения и вложенного шага пайплайна к единому запуску. Подробности о строковых лямбда-выражениях см. в [руководстве](ЛямбдаВыражения.md).
