# КонтекстПереполненияПараллелизма

## Синтаксис:

```bsl
Новый КонтекстПереполненияПараллелизма(<ЛимитПараллелизма>, <ЛимитОчереди>, <КоличествоВыполняющихся>, <КоличествоОжидающих>)
```

**Параметры:**

| Имя | Тип | Описание |
| -- | -- | -- |
| ЛимитПараллелизма | Число | Максимально допустимое количество одновременно выполняющихся операций |
| ЛимитОчереди | Число | Максимально допустимое количество ожидающих операций |
| КоличествоВыполняющихся | Число | Количество операций, выполнявшихся в момент переполнения |
| КоличествоОжидающих | Число | Количество операций, ожидавших свободный слот в момент переполнения |
