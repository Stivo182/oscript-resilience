# КонтекстПопыткиПовтора

## Синтаксис:

```bsl
Новый КонтекстПопыткиПовтора(<НомерПопытки>, <ПопытокОсталось>, <Результат>, <ЕстьИсключение>, <ДанныеОшибки>, <ДлительностьВыполнения>, <Задержка>)
```

**Параметры:**

| Имя | Тип | Значение по умолчанию | Описание |
| -- | -- | -- | -- |
| НомерПопытки | Число |  | Номер текущей попытки, начиная с `0` |
| ПопытокОсталось | Число | `0` | Количество повторных попыток, доступных после текущей |
| Результат | Произвольный | `Неопределено` | Результат операции |
| ЕстьИсключение | Булево | `Ложь` | Признак завершения попытки с исключением |
| ДанныеОшибки | ИнформацияОбОшибке, Неопределено | `Неопределено` | Информация об ошибке |
| ДлительностьВыполнения | Число | `0` | Длительность выполнения попытки в миллисекундах |
| Задержка | Число | `0` | Задержка перед следующим повтором в миллисекундах |
