# КонтекстРазмыканияПредохранителя

## Синтаксис:

```bsl
Новый КонтекстРазмыканияПредохранителя(<ПорогОшибок>, <КоличествоОшибокПодряд>, <ДлительностьРазмыкания>, <ДанныеОшибки>, <Состояние>)
```

**Параметры:**

| Имя | Тип | Описание |
| -- | -- | -- |
| ПорогОшибок | Число | Количество ошибок подряд до размыкания |
| КоличествоОшибокПодряд | Число | Текущее количество ошибок подряд |
| ДлительностьРазмыкания | Число | Длительность размыкания в миллисекундах |
| ДанныеОшибки | ИнформацияОбОшибке | Ошибка, вызвавшая размыкание |
| Состояние | Строка | Текущее состояние предохранителя |

## ПорогОшибок

**Синтаксис:**

```bsl
ПорогОшибок()
```

**Возвращаемое значение:**

Тип: Число.

**Описание:**

Возвращает порог ошибок подряд до размыкания.
