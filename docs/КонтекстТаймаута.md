# КонтекстТаймаута

## Синтаксис:

```bsl
Новый КонтекстТаймаута(<Таймаут>, <ДлительностьВыполнения>)
```

**Параметры:**

| Имя | Тип | Описание |
| -- | -- | -- |
| Таймаут | Число | Настроенное значение таймаута в миллисекундах |
| ДлительностьВыполнения | Число | Фактическая длительность ожидания до срабатывания таймаута в миллисекундах |


## Методы

[Таймаут](#таймаут) </br>
[ДлительностьВыполнения](#длительностьвыполнения)


## Таймаут

**Синтаксис:**

```bsl
Таймаут()
```

**Возвращаемое значение:**

Тип: Число.

**Описание:**

Возвращает настроенное значение таймаута в миллисекундах.


## ДлительностьВыполнения

**Синтаксис:**

```bsl
ДлительностьВыполнения()
```

**Возвращаемое значение:**

Тип: Число.

**Описание:**

Возвращает фактическую длительность ожидания до срабатывания таймаута в миллисекундах.
