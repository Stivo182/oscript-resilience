# Лямбда-выражения

Строковые лямбда-выражения выполняются через пакет [`lambdas`](https://github.com/sfaqer/lambdas).

Их можно использовать:

- в обработчиках стратегий
- как пользовательские операции для `Запустить(...)`, `ЗапуститьАсинх(...)` и `Применить(...)`

## Лямбды в обработчиках

Обычно такая лямбда получает первым аргументом контекст соответствующей стратегии. После контекста можно передать дополнительные параметры через публичный API стратегии или пайплайна. Тип возвращаемого значения зависит от конкретного метода.

Если обработчику нужна побочная логика, например логирование или накопление данных, удобнее передавать `Действие`.

### Примеры

```bsl
СтратегияПовтора.УстановитьУсловиеПовтора(
    "(Контекст) -> Возврат Контекст.ЕстьИсключение()"
);

ПовторятьПриОшибке = Истина;
СтратегияПовтора.УстановитьУсловиеПовтора(
    "(Контекст, ПовторятьПриОшибке) -> Возврат Контекст.ЕстьИсключение() И ПовторятьПриОшибке",
    ПовторятьПриОшибке
);

СтратегияПовтора.УстановитьВычислениеЗадержки(
    "(Контекст) -> Возврат Контекст.НомерПопытки() * 100"
);

СтратегияЗапасногоСценария.УстановитьОбработчик(
    "(Контекст) -> Возврат ""Резервный результат"""
);
```

## Лямбды как операции пайплайна

Если в `Запустить(...)`, `ЗапуститьАсинх(...)` или `Применить(...)` передано лямбда-выражение, библиотека создает из него пользовательскую операцию и передает в нее `Параметры`:

- `Неопределено` передает в лямбду ноль аргументов
- одиночное значение передает один аргумент
- `Массив` или `ФиксированныйМассив` передает аргументы по позициям

```bsl
ПараметрыОперации = Новый Массив();
ПараметрыОперации.Добавить("left");
ПараметрыОперации.Добавить("right");

Пайплайн.Запустить(
    "(ПервыйПараметр, ВторойПараметр) -> Возврат СтрШаблон(""%1:%2"", ПервыйПараметр, ВторойПараметр)",
    ПараметрыОперации
);

Обещание = Пайплайн.ЗапуститьАсинх(
    "(Имя) -> Возврат СтрШаблон(""Привет, %1!"", Имя)",
    "Мир"
);

Результат = Обещание.Получить();
```

Методы `Получить()` и `Поймать()` относятся к объекту `Обещание` из библиотеки [async](https://github.com/sfaqer/async/).

Если асинхронная операция завершается с ошибкой, `Получить()` выбрасывает `ИнформацияОбОшибке`, а `Поймать()` позволяет обработать или преобразовать ошибку.
