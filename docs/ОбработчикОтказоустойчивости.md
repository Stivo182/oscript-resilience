# ОбработчикОтказоустойчивости

## Синтаксис:

```bsl
Новый ОбработчикОтказоустойчивости(<Обработчик>, <ДополнительныеПараметры>, <ЭтоФункция>)
```

**Параметры:**

| Имя | Тип | Значение по умолчанию | Описание |
| -- | -- | -- | -- |
| Обработчик | Действие, Строка |  | Действие или лямбда-выражение |
| ДополнительныеПараметры | Массив, ФиксированныйМассив, Произвольный, Неопределено | `Неопределено` | Дополнительные параметры, которые будут переданы обработчику после контекста |
| ЭтоФункция | Булево | `Истина` | Признак того, что обработчик возвращает значение |

## Вызвать

**Синтаксис:**

```bsl
Вызвать(<Контекст>)
```

**Параметры:**

| Имя | Тип | Описание |
| -- | -- | -- |
| Контекст | Произвольный | Контекст вызова обработчика |

**Возвращаемое значение:**

Тип: Произвольный.

**Описание:**

Вызывает подготовленный обработчик. Сначала в обработчик передается `Контекст`, затем дополнительные параметры, указанные при создании экземпляра. Подробности о лямбда-выражениях см. в [руководстве](ЛямбдаВыражения.md).
