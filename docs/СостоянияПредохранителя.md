# СостоянияПредохранителя

## Синтаксис:

```bsl
СостоянияПредохранителя.<Значение>
```

## Значения

[Закрыт](#закрыт) </br>
[Разомкнут](#разомкнут) </br>
[Полуразомкнут](#полуразомкнут)


## Закрыт

**Синтаксис:**

```bsl
СостоянияПредохранителя.Закрыт
```

**Тип значения:**

Строка.

**Описание:**

Вызовы разрешены, предохранитель работает в штатном режиме.


## Разомкнут

**Синтаксис:**

```bsl
СостоянияПредохранителя.Разомкнут
```

**Тип значения:**

Строка.

**Описание:**

Новые вызовы временно запрещены.


## Полуразомкнут

**Синтаксис:**

```bsl
СостоянияПредохранителя.Полуразомкнут
```

**Тип значения:**

Строка.

**Описание:**

Предохранитель разрешает пробную попытку после окончания времени размыкания.
