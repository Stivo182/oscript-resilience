**Английское название:** `Bulkhead`.

## Синтаксис:

```bsl
Новый СтратегияОграниченияПараллелизма(<ЛимитПараллелизма>, <ЛимитОчереди>)
```

**Параметры:**

| Имя | Тип | Значение по умолчанию | Описание |
| -- | -- | -- | -- |
| ЛимитПараллелизма | Число |  | Максимально допустимое количество одновременно выполняющихся операций |
| ЛимитОчереди | Число | `0` | Максимально допустимое количество операций, ожидающих свободный слот |

## Применить

**Синтаксис:**

```bsl
Применить(<Операция>, <Параметры>)
```

**Параметры:**

| Имя | Тип | Значение по умолчанию | Описание |
| -- | -- | -- | -- |
| Операция | Действие, ШагПайплайнаОтказоустойчивости, Строка |  | Выполняемая операция |
| Параметры | Массив, ФиксированныйМассив, Произвольный, Неопределено | `Неопределено` | Параметры операции |

**Возвращаемое значение:**

Тип: Произвольный.

**Описание:**

Выполняет операцию, если в стратегии есть свободный слот параллелизма. Если все слоты заняты, операция может быть поставлена в очередь ожидания в пределах `ЛимитОчереди`.


## УстановитьЛимитПараллелизма

**Синтаксис:**

```bsl
УстановитьЛимитПараллелизма(<ЛимитПараллелизма>)
```

**Параметры:**

| Имя | Тип | Описание |
| -- | -- | -- |
| ЛимитПараллелизма | Число | Максимально допустимое количество одновременно выполняющихся операций |

**Возвращаемое значение:**

Тип: СтратегияОграниченияПараллелизма.

**Описание:**

Устанавливает лимит параллелизма. Метод предназначен для настройки стратегии до начала выполнения операций.


## УстановитьЛимитОчереди

**Синтаксис:**

```bsl
УстановитьЛимитОчереди(<ЛимитОчереди>)
```

**Параметры:**

| Имя | Тип | Описание |
| -- | -- | -- |
| ЛимитОчереди | Число | Максимально допустимое количество ожидающих операций |

**Возвращаемое значение:**

Тип: СтратегияОграниченияПараллелизма.

**Описание:**

Устанавливает лимит очереди ожидания. Значение `0` означает немедленное отклонение операции при отсутствии свободного слота.


## УстановитьОбработчикПереполнения

**Синтаксис:**

```bsl
УстановитьОбработчикПереполнения(<Обработчик>, <ДополнительныеПараметры>)
```

**Параметры:**

| Имя | Тип | Описание |
| -- | -- | -- |
| Обработчик | Действие, Строка | Обработчик, вызываемый при переполнении лимита параллелизма. Строка трактуется как лямбда-выражение. Получает [КонтекстПереполненияПараллелизма](КонтекстПереполненияПараллелизма.md). Возвращаемое значение не используется |
| ДополнительныеПараметры | Массив, ФиксированныйМассив, Произвольный, Неопределено | Дополнительные параметры, которые будут переданы обработчику после контекста переполнения |

**Возвращаемое значение:**

Тип: СтратегияОграниченияПараллелизма.

**Описание:**

Устанавливает обработчик переполнения лимита параллелизма. Возвращаемое значение обработчика игнорируется. Если заданы дополнительные параметры, они передаются после контекста переполнения. Подробности о лямбда-выражениях см. в [руководстве](ЛямбдаВыражения.md).
