| Имя | Тип | Описание |
| -- | -- | -- |
| Таймаут | Число | Максимально допустимое время ожидания в миллисекундах |

## Применить

**Синтаксис:**

```bsl
Применить(<Операция>, <Параметры>)
```

**Параметры:**

| Имя | Тип | Значение по умолчанию | Описание |
| -- | -- | -- | -- |
| Операция | Действие, ШагПайплайнаОтказоустойчивости |  | Выполняемая операция |
| Параметры | Массив, Неопределено | `Неопределено` | Параметры операции |

**Возвращаемое значение:**

Тип: Произвольный.

**Описание:**

Выполняет операцию и ожидает завершения не дольше заданного таймаута.


## Таймаут

**Синтаксис:**


**Возвращаемое значение:**

**Описание:**

Возвращает установленный таймаут в миллисекундах.
