# ТипыРасчетаЗадержки

## Синтаксис:

```bsl
ТипыРасчетаЗадержки.<Значение>
```

## Значения

[Постоянная](#постоянная) </br>
[Линейная](#линейная) </br>
[Экспоненциальная](#экспоненциальная)


## Постоянная

**Синтаксис:**

```bsl
ТипыРасчетаЗадержки.Постоянная
```

**Тип значения:**

Строка.

**Описание:**

Задержка не меняется между повторными попытками.


## Линейная

**Синтаксис:**

```bsl
ТипыРасчетаЗадержки.Линейная
```

**Тип значения:**

Строка.

**Описание:**

Задержка растет линейно от номера попытки.


## Экспоненциальная

**Синтаксис:**

```bsl
ТипыРасчетаЗадержки.Экспоненциальная
```

**Тип значения:**

Строка.

**Описание:**

Задержка растет экспоненциально от номера попытки.
