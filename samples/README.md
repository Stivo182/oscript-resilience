# Примеры

В каталоге `samples` собраны небольшие сценарии, которые показывают библиотеку в действии.
Каждый пример можно запускать отдельно и использовать как отправную точку для собственных экспериментов.

## Как запускать

Из корня репозитория:

```bash
oscript samples/ПримерОтказоустойчивости.os
```

По тому же шаблону можно запускать любой другой файл из каталога `samples`.

## Что есть в каталоге

- [ПримерОтказоустойчивости.os](ПримерОтказоустойчивости.os) показывает запуск одного действия через пайплайн из нескольких стратегий.
- [ПримерСтратегииПовтора.os](ПримерСтратегииПовтора.os) показывает повтор операции после временных ошибок.
- [ПримерСтратегииПредохранителя.os](ПримерСтратегииПредохранителя.os) показывает размыкание предохранителя и пробный вызов после ожидания.
- [ПримерСтратегииТаймаута.os](ПримерСтратегииТаймаута.os) показывает ошибку по таймауту и повторный запуск после увеличения лимита ожидания.
- [ПримерСтратегииОграниченияВызовов.os](ПримерСтратегииОграниченияВызовов.os) показывает ограничение количества вызовов в интервале.
- [ПримерСтратегииЗапасногоСценария.os](ПримерСтратегииЗапасногоСценария.os) показывает возврат запасного результата вместо проброса ошибки.

## Вспомогательные классы

- [Классы/НестабильныйСервис.os](Классы/НестабильныйСервис.os) имитирует сервис, который некоторое количество раз завершается ошибкой, а затем начинает отвечать успешно.
- [Классы/МедленныйСервис.os](Классы/МедленныйСервис.os) имитирует сервис с искусственной задержкой, удобный для примеров со стратегией таймаута.

## С чего начать

Если хочется быстро увидеть библиотеку целиком, начни с [ПримерОтказоустойчивости.os](ПримерОтказоустойчивости.os).
Если нужен конкретный сценарий, удобнее открыть пример нужной стратегии и запускать его отдельно.
